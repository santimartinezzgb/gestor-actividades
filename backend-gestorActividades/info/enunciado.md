APP DE PLATAFORMA DE GESTIÓN DE ACTIVIDADES

# Enunciado
Back spring boot + java + mongodb
Front vuejs + electron
Front movil flutter + material designs

Roles
- Administrador. crea modifica y elimina actividades. Ve reservas y controla asistencia
- Usuario. inicia sesion y se registra. Ve, reserva y cancela actividades. Consulta sus reservas

Funcionalidades principales
- Actividades crud completo solo admin
- reservar -> evitar dpulicados , cancelación 15 mins antes y admin verifica asistencia
- validaciones en backend
