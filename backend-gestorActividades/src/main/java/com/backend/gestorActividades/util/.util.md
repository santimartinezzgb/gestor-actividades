# ¿Directorio util?
El paquete `util` contiene utilidades y helpers que son usadas por distintos componentes de la aplicación (validaciones, utilidades de conversión, etc).

Qué incluir aquí:
- Métodos de validación reutilizables
- Clases helper (por ejemplo `ValidationUtil`)
- Funciones puramente utilitarias que no pertenecen a una entidad o servicio concreto

Buenas prácticas:
- Mantener utilidades genéricas y bien documentadas
- Evitar convertir utilidades en "cajón de sastre"; si una utilidad está muy específica, muévela al paquete que corresponda
