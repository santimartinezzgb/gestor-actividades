# ¿Directorio exception?
El paquete `exception` agrupa las excepciones personalizadas de la aplicación y los manejadores globales.

Qué puedes encontrar:
- Excepciones específicas del dominio (por ejemplo `DuplicateReserveException`, `DuplicateUserException`)
- `GlobalExceptionHandler` con `@ControllerAdvice` para mapear excepciones a respuestas HTTP

Buenas prácticas:
- Definir excepciones claras y específicas para los casos de error del dominio
- Centralizar el mapeo a códigos HTTP en un `@ControllerAdvice`
- Evitar lanzar `RuntimeException` genéricas cuando se puede usar una excepción específica
