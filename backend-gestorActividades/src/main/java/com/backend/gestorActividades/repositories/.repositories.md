# ¿Directorio repositories?
El paquete `repositories` contiene interfaces Spring Data que actúan como la capa de acceso a datos.

Qué encontrarás:
- Interfaces que extienden `MongoRepository` o `CrudRepository` para entidades (`UserRepository`, `ActivityRepository`, `ReserveRepository`)
- Consultas derivadas por nombre o `@Query` cuando sea necesario

Buenas prácticas:
- Mantener las consultas y proyecciones necesarias para la lógica de negocio
- Evitar lógica de negocio compleja en los repositorios; que solo manejen acceso y consultas
