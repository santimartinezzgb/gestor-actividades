# ¿Directorio DTO?
DTO significa "Data Transfer Object" (Objeto de Transferencia de Datos).

Es un patrón de diseño utilizado para transferir datos entre diferentes capas de una aplicación, especialmente entre la capa de presentación y la capa de negocio.

En el contexto de una app Spring Boot, los DTOs se utilizan comúnmente para representar los datos que se envían y reciben a través de las API REST. Estos objetos suelen ser simples POJOs (Plain Old Java Objects) que contienen campos y métodos getter/setter, pero no contienen lógica de negocio.
