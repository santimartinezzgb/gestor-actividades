# ¿Directorio security?
El paquete `security` contiene la configuración de seguridad de la aplicación: servicios y clases relacionadas con Spring Security.

## ¿Qué es un Bean?
Un **Bean** es un objeto gestionado por Spring. Se define con `@Bean` en clases de configuración y permite que Spring lo cree, configure e inyecte donde se necesite.

## ¿Para qué se usa?
- **Reutilización**: Un mismo objeto compartido en toda la aplicación.
- **Configuración centralizada**: Define componentes como `PasswordEncoder` o `SecurityFilterChain` una vez.
- **Inyección de dependencias**: Spring inyecta automáticamente estos beans donde se necesiten usando `@Autowired`.
