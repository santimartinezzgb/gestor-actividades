# ¿Directorio security?
El paquete `security` contiene la configuración de seguridad de la aplicación: servicios y clases relacionadas con Spring Security.

Qué puedes encontrar:
- `SecurityConfig` con la definición de `SecurityFilterChain`
- Filtros personalizados (por ejemplo `JwtAuthenticationFilter`)
- Servicios relacionados con autenticación/autorization (por ejemplo `JwtService`, `UserDetailsServiceImpl`)

Notas:
- Se ha eliminado el manejo de tokens JWT en favor de una implementación más simple que usa Spring Security y datos de usuario en la base de datos.
- Si se desea reintroducir JWT en el futuro, habría que añadir un servicio para creación/validación de tokens, un filtro de autenticación y las dependencias correspondientes (por ejemplo jjwt). Actualmente el proyecto no usa esas clases ni dependencias.
