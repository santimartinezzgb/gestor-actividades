# ¿Directorio security?
El paquete `security` contiene la configuración de seguridad de la aplicación: filtros, servicios para JWT y clases relacionadas con Spring Security.

Qué puedes encontrar:
- `SecurityConfig` con la definición de `SecurityFilterChain`
- Filtros personalizados (por ejemplo `JwtAuthenticationFilter`)
- Servicios relacionados con autenticación/autorization (por ejemplo `JwtService`, `UserDetailsServiceImpl`)

Notas:
- Mantener la lógica de manipulación de tokens y autenticación aquí.
- Preferir inyectar dependencias por interfaz para mantener flexibilidad.
