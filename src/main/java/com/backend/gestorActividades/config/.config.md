# ¿Directorio config?
El paquete `config` contiene clases de configuración de la aplicación. Aquí se colocan componentes que inicializan o configuran beans globales, servicios de integración, o adaptadores del framework.

Ejemplos típicos que puedes encontrar:
- Configuración de persistencia, inicializadores de datos (DataInitializer)
- Configuraciones específicas de Mongo o de otros servicios externos
- Definiciones de beans auxiliares que deben ser visibles a toda la aplicación

Nota: Evita colocar lógica de negocio en este paquete; su responsabilidad es configurar y proveer beans y parámetros de integración.
